**Cartopy**

- `Python` 에서 지리공간 데이터를 지도 위에 시각화 할 수 있도록하는 데이터 시각화 라이브러리
  —> 위도 경도 데이터를 실제 지도 형태로 정확하게 표현하고 싶을 때 사용하는 도구
- `matplotlib` 을 기반으로 구축됨
  —> `matplotlib` 의 그래프 기능을 그대로 활용하면서 지리적 데이터 시각화가 가능함.
- `PROJ` , `numpy` , `Shapely` 라이브러리 활용

**Cartopy를 사용하는 이유**

- 지구 곡률을 고려한 지도 투영이 가능함. (Projection)
- 국가 경계선, 해안선 자동 추가
- 위경도 좌표 변환 자동 처리
- 기상, 해양, 인구, 교통 데이터 시각화 가능

**주요 특징**

- large area (대륙 단위) / small scale data 에서 유용함.
  \*지도학에서 `Small scale map` 은 넓은 지역 (= 세계지도), `Large scale map` 은 좁은 지역(= 동네 지도)
