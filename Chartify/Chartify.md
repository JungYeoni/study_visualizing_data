**Chartify**

- Spotify에서 개발한 `python` 데이터 시각화 라이브러리
- `Bokhe` 를 기반 엔진으로 사용함.
- 직관적인 API 제공을 통해 차트를 쉽고 빠르게 제작할 수 있는 웹 서비스 (인터랙티브 차트)

**개발 이유**

- `matplotlib` 은 유연하지만 디자인이 통일되지 않음
- `seabornd` 은 예쁘지만 대규모 비즈니스 리포트용으로 일관성 유지가 어려움
- `Bokeh` 는 강력하지만 low-level API라 코드가 길어짐.

—> 일관된 스타일 + 직관적 API + 인터랙티브 기능을 동시에 만족하는 wrapper 라이브러리 필요

**주요 기능**

- 다양한 차트를 지원함.
- 사용자의 요구에 따라 커스터마이징이 자유로움
- 데이터를 직접 처리하고 변환하는 데 필요한 기능을 내장하고 있음

**특징**

- 장점
  - 코드가 간결하고 기본 디자인이 우수함
  - `pandas` 와 자연스럽게 통합
- 단점
  - 커뮤니티가 크지 않음 → 최신 업데이트 빈도 낮음
  - 고급 커스터마이징에는 결국 `Bokeh` 이해가 필요함
  - `seaborn` 처럼 통계적 시각화에 특화되진 x

**—> 언제 사용하면 좋을까?**

- 내부 리포트를 자동화하고 싶을 때
- 반복 보고서를 생성할 때
- 마케팅/비즈니스 대시보드
